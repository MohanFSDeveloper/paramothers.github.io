Sheet 3: Selector syntax
based on position







a:first
select first <a> element in page or given context

a:last
select last <a> element in a page or given context

a:odd
select all odd <a> tag in a page or given context

p:even
select all even <p> tag in page or given context

p:only-child
select <p> tag if it has only child but not siblings

ul li:last-child
select last <li> of each <ul> in a page

tr:nth-child(1/odd/even/5n+1)
select EVERY first row in all tables or odd or even or after 5th row

tr:eq(2)
return every 2nd row of all tables in page

tr:gt(2)
return every greather than 2 row of all tables in page

tr:lt(2)
return every less than 2 row of all tables in page

based on attribute name/ value







form[method]
it select <form> element , who has 'method' attribute regardless of its value

a[href$= 'pdf']
select<a> element whose href attribute value end with 'pdf'

a[href^= 'http://']
select<a> element whose href attribute value start with 'http://'

a[href*='googl.com']
it select all <a>, it has attribute href and its value has 'googl.com' anywhere.

input[type !='text']
it select <input> tag, Either it has type attribute with value 'text' or doest not have type attribute itself

input[type='text']
it select all <input> element, whose attribute name type and its value 'text', usually it is text box
