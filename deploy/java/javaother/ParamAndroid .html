Sheet 3: Tools
Android Studio it is IDE, built on top of Intelli J IDE
Android Platform it is native system, that is linux kernal, application framework
Application Framework Layer it the java API, a application developer mostly use.
ADT Bundle it is for Android development plugin for eclipse.
JDK Java Development Kit for application Development
Android SDK it is like JDK, used for Android development, testing application, Emulator
Dalvik Convert java byte code to dex instruction ( Dalvik Virtual Machine )
APK Android Application Package
aapt Tool, to generate R.java file
apkbuilder it package a application with its resource, dex, xml files as apk file. Apk file just like jar or zip file
emulator it is capbable of running any number of Android Virtual Devices ( AVD )
AVD tool used to create AVDs for testing android application


LogCat display the log message, but many features


DDMS Dalvik Debug Monitor Server, it has set of debugging tools


ADB Android Debug Bridge, used to communicate with an emulator or connected Android device


CTS Compartibility Test Suite, what are the devices are suitable for android, if not, they cannot use Google Play store
