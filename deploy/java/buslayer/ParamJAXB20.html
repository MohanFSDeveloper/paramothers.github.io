Sheet 4: Step to customize

Algrorithmic Steps of JAXB Customization


1 Finish XML Schema
2 write ANT script for compilation of xml Schema into java bean
3 customize the <globalBindings>
4 customize for <schemaBinding> for package
5 customize class name
6 check the mapping of xml schema elements with bean for every getting inner level of element
7 write JavaDoc for java classes
8 customize property name of class [ use any attiribute like generateIsSetMethod set to false <property> element if necessary . Better use globalbinding]
9 use <javaType> to change default Data type mapping between java and xml
10 write customized data type binding methds for wrapper. It can reduce creating BIG<Wrapper> objects. Mostly customize with built in methods instead of writing own class and methods
11 customize simpleType for java enumeration if a simple type has only Enumeration facets
13 customize <xmlNameTransform> with <elementName> with Suffix value as "Element". It will increase readability of bean. Mostly root Bean will end with this Element
15 if you want to javadoc should comes the JAXB binding tag palce all the binding element inside of <!CDATA[ ]] >
