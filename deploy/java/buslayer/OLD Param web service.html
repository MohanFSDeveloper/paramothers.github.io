Sheet 9: order of webservice development
1 Write a XMLSchema
2 write JAXB EXTERNAL customization for given xmlSchema
3 write a ANT script to generate JAVA bean for this xmlSchema
4 write a Java INTERFACE using these java bean as paremeter or return type
5 write WSDL file based on this interface and importing that prior developed xmlSchema
6 write Mapping file
7 write JSE or EJB enpoints
8 write web.xml / ejb-jar.xml file
9 write webservices.xml
10 write server specific webservices.xml file
11 now do any security or transaction or any other configuration in web.xml or ejb-jar.xml as advanced settings









alternative order



write interface, which declare methods

write implmentation class

write web.xml

write XMLSchema

write wsdl

write mapping file

write webservices.xml
