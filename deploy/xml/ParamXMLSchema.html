11: Import, Include, Redefine
import allow to build a schema document from multiple other schema document with different namespace.

it cannot override for any those imported schema components ( element / type /attributte / model group definition )




include allow to build a schema document from multiple other schema document ( without different name space )

it adopt the same namespace of including schema component

it can redefine any of schema component of included schema document

the value of SCHEMALOCATION attribute must be valid. If not validator does not consider this include of schema.. Simply ignore
